# 初三法语词汇收集研究计划

## 任务概述
研究收集适合初三学生学习的法语词汇，包括社会生活、文化旅游、复杂动词形式和形容词比较级最高级，目标80-120个词汇。

## 任务步骤

### 1. 基础研究阶段
- [x] 1.1 了解初三学生法语学习水平和标准
- [x] 1.2 研究CEFR（欧洲语言共同参考框架）对应水平
- [x] 1.3 分析初三法语课程标准和要求

### 2. 词汇分类研究
- [x] 2.1 社会生活词汇研究
  - [x] 家庭和学校生活
  - [x] 朋友和社交活动
  - [x] 日常生活用品和服务
- [x] 2.2 文化、旅游词汇研究
  - [x] 文化活动和节日
  - [x] 旅行和交通
  - [x] 地理和城市名称
- [x] 2.3 复杂动词形式和语法结构词汇研究
  - [x] 不规则动词
  - [x] 复合时态词汇
  - [x] 连接词和介词
- [x] 2.4 形容词比较级和最高级研究
  - [x] 规则形容词变位
  - [x] 不规则形容词形式

### 3. 词汇验证和收集
- [x] 3.1 使用权威法语词典验证词汇
- [x] 3.2 确认音标准确性
- [x] 3.3 检查词性标注
- [x] 3.4 收集目标词汇（110个，符合80-120个要求）

### 4. 数据整理和输出
- [x] 4.1 按类别整理词汇
- [x] 4.2 转换为JSON格式
- [x] 4.3 保存到data/grade9_words.json文件
- [x] 4.4 最终质量检查

## 目标词汇数量分配
- 社会生活词汇：25-30个
- 文化、旅游词汇：25-30个  
- 复杂动词形式和语法结构词汇：20-25个
- 形容词比较级和最高级：10-15个
- 总计：80-120个词汇

## 质量标准
- 每个词汇必须有：法语原文、中文释义、词性、发音音标
- 词汇适合初三学生水平
- 准确性通过权威来源验证
- JSON格式符合要求
