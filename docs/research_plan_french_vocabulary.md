# 法语基础词汇收集研究计划

## 研究目标
收集适合初一学生学习的法语基础词汇80-120个，包含法语原文、中文释义、词性、发音。

## 任务类型
**搜索-收集任务**：重点在于全面收集基础词汇，确保内容的准确性和适合性。

## 研究范围
1. **日常生活词汇**
   - [x] 家庭成员 (10个词汇)
   - [x] 学校用品和场景 (20个词汇)
   - [x] 食物和饮料 (10个词汇)
   - [x] 颜色 (16个词汇)
   - [x] 数字0-19 (20个词汇)

2. **常见动词**
   - [x] 基础动作动词 (25个词汇)
   - [x] 日常使用动词 (15个词汇)

3. **基础语法词汇**
   - [x] 冠词和介词 (8个词汇)
   - [x] 形容词 (10个词汇)

## 数据验证要求
- [x] 验证法语发音准确性
- [x] 确认词汇适合初一学生水平
- [x] 检查中文释义准确性
- [x] 标准化词性分类

## 最终输出
- [x] 将收集的词汇保存到data/grade7_words.json
- [x] 格式：JSON数组，每项包含french、chinese、phonetic、part_of_speech字段
