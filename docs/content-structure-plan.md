# 内容结构计划 - 初中法语背单词移动应用

## 1. 材料清单
**数据文件：**
- `data/grade7_words.json` (105个词汇，初一)
- `data/grade8_words.json` (100个词汇，初二)
- `data/grade9_words.json` (110个词汇，初三)
- 总计：315个词汇
- 字段：french（法语）、chinese（中文）、phonetic（音标）、part_of_speech（词性）

**文档资料：**
- `docs/french_vocabulary_report.md` (词汇研究报告)

## 2. 应用结构
**类型：** 移动端单页应用（SPA）
**理由：** 
- 核心功能集中（学习、列表、统计）
- 需要流畅的页面切换体验
- 状态管理需要持久化（学习进度、已掌握单词）
- 适合移动端左右滑动交互

## 4. 内容分析
**信息密度：** 中等
- 数据：315个词汇条目，每个4个字段
- 状态数据：每个词汇的掌握状态（布尔值）
- 统计数据：实时计算的进度和掌握率

**内容平衡：**
- 数据驱动：100%（词汇数据 + 状态数据）
- 无图片素材（纯文本学习应用）
- 内容类型：数据密集型

**交互特性：**
- 高频交互：标记"已掌握/未掌握"（每学习一个单词）
- 中频交互：切换年级、切换界面、筛选列表
- 低频交互：查看统计、设置偏好

**移动端优化重点：**
- 垂直滚动优先
- 大按钮触摸区域（≥56px）
- 单手操作友好的底部导航
- 卡片翻转/滑动动画增强学习体验
