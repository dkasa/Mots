## 🚀 PWA 是什么？
PWA（Progressive Web App）是一种结合了网页和原生应用优势的新型应用。用户可以将PWA"安装"到设备主屏幕，使用体验与原生应用几乎相同，但无需通过应用商店下载。

### PWA 的优势
- 📱 **像原生应用**: 安装后可像手机应用一样使用
- 🌐 **无需应用商店**: 直接通过浏览器访问和安装
- 💾 **离线使用**: 支持离线学习，断网也能用
- 🔄 **自动更新**: 应用会自动更新到最新版本
- 📊 **占用空间小**: 比原生应用占用更少存储空间
- 🔒 **安全可靠**: 基于HTTPS，数据传输安全

## 📲 安装指南

### Android 设备 (推荐 Chrome 浏览器)

#### 方法一：自动安装提示
1. **打开应用**: 使用 Chrome 访问 https://yuvtex7gwoe1.space.minimaxi.com
2. **等待提示**: 30秒后，屏幕底部会出现绿色安装横幅
3. **点击安装**: 点击横幅中的"安装"按钮
4. **确认安装**: 在弹出的"添加到主屏幕"对话框中点击"安装"
5. **完成**: 应用图标将出现在主屏幕上，可以像其他应用一样使用

#### 方法二：手动安装
1. **打开应用**: 使用 Chrome 访问应用地址
2. **菜单操作**: 点击浏览器右上角的三点菜单（⋮）
3. **添加到主屏幕**: 选择"添加到主屏幕"选项
4. **确认安装**: 在对话框中点击"安装"

### iOS 设备 (使用 Safari 浏览器)

1. **打开应用**: 使用 Safari 访问 https://yuvtex7gwoe1.space.minimaxi.com
2. **打开分享菜单**: 点击底部工具栏的分享按钮（方框带向上箭头图标）
3. **添加到主屏幕**: 在分享菜单中向下滚动，找到"添加到主屏幕"选项并点击
4. **自定义名称**: 可以编辑应用名称（默认为"法语背单词"），然后点击"添加"
5. **完成**: 应用图标将出现在主屏幕上

### Windows 电脑 (Chrome/Edge 浏览器)

1. **打开应用**: 使用 Chrome 或 Edge 访问 https://yuvtex7gwoe1.space.minimaxi.com
2. **地址栏提示**: 地址栏右侧可能会出现安装图标（➕）
3. **点击安装**: 点击安装图标或地址栏出现的"安装法语背单词"提示
4. **确认安装**: 在弹出的对话框中点击"安装"
5. **完成**: 应用将出现在开始菜单和任务栏，可以像桌面应用一样使用

### macOS 电脑 (Safari 浏览器)

1. **打开应用**: 使用 Safari 访问 https://yuvtex7gwoe1.space.minimaxi.com
2. **菜单栏**: 点击菜单栏的"文件"选项
3. **添加到 Dock**: 选择"添加到 Dock"选项
4. **完成**: 应用图标将出现在 Dock 中，可以像其他应用一样使用

## 🎯 使用指南

### 首次使用
1. **启动应用**: 点击主屏幕上的应用图标启动
2. **观看启动**: 应用会显示精美的启动画面（2秒）
3. **开始学习**: 选择年级（初一/初二/初三），开始法语单词学习

### 主要功能

#### 📚 单词学习模式
- **年级选择**: 点击顶部Tab切换不同年级
- **单词卡片**: 查看法语单词、音标、中文释义和词性
- **标记掌握**: 点击"已掌握"或"未掌握"按钮
- **自动切换**: 操作后自动进入下一个单词
- **进度跟踪**: 顶部显示学习进度和掌握情况

#### 📋 单词列表模式  
- **切换模式**: 点击底部导航的"列表"按钮
- **筛选查看**: 使用筛选器查看"全部"、"已掌握"或"未掌握"单词
- **下拉刷新**: 在列表顶部下拉可刷新数据
- **快速学习**: 点击任意单词可快速进入该单词的学习界面

#### 📊 学习统计
- **实时进度**: 显示当前年级已学/总数、已掌握数量
- **进度条**: 可视化的学习进度百分比
- **数据持久化**: 学习进度自动保存，刷新不丢失

### 💡 使用技巧

#### 学习效率
1. **专注模式**: 建议在安静环境下专注学习
2. **定时学习**: 建议每次学习15-30分钟
3. **复习巩固**: 已掌握的单词会减少重复出现
4. **循序渐进**: 按年级顺序学习，基础更扎实

#### 移动端优化
1. **横屏使用**: 支持横屏使用，界面自动适配
2. **触摸优化**: 按钮按压有缩放反馈效果
3. **滑动操作**: 在单词卡片上可左右滑动快速操作
4. **离线使用**: 断网后仍可正常学习

#### 数据管理
1. **自动保存**: 学习进度自动保存到本地
2. **数据备份**: 重要学习数据建议截图保存
3. **重置功能**: 可在设置中重置所有学习数据
4. **跨设备**: 同一浏览器内的数据会保持一致

## 🔧 常见问题

### 安装相关

**Q: 为什么没有出现安装提示？**
A: 安装提示会在访问30秒后出现。如果您已经安装过应用或拒绝了安装，提示可能不会再次显示。您可以尝试清除浏览器数据后重新访问。

**Q: 安装后图标显示不正确怎么办？**
A: 这是浏览器缓存问题。请尝试：
1. 卸载应用（长按图标选择"卸载"）
2. 清除浏览器缓存
3. 重新访问并安装应用

**Q: 在哪些浏览器中可以安装？**
A: 支持安装的浏览器：
- Android: Chrome、Edge、Firefox
- iOS: Safari
- Windows: Chrome、Edge
- macOS: Safari、Chrome

### 使用相关

**Q: 离线时应用无法使用？**
A: 请确保：
1. Service Worker 已正确注册（首次访问时会自动安装）
2. 网络连接正常时已成功加载过应用
3. 清除浏览器数据后重新加载应用

**Q: 学习进度丢失怎么办？**
A: 学习数据保存在本地浏览器中，清理浏览器数据会清除进度。建议：
1. 不要随意清理浏览器数据
2. 定期截图保存重要学习统计
3. 使用同一设备/浏览器保持数据连续性

**Q: 应用更新后功能异常？**
A: PWA应用会自动更新。如果遇到问题：
1. 强制刷新页面（Ctrl+F5 或 Ctrl+Shift+R）
2. 重新启动浏览器
3. 清除应用缓存后重新访问

### 性能相关

**Q: 应用运行缓慢怎么办？**
A: 优化建议：
1. 关闭其他不必要的浏览器标签页
2. 重启浏览器或设备
3. 清除浏览器缓存和临时文件
4. 确保设备存储空间充足

**Q: 字体显示异常？**
A: 字体问题可能由以下原因造成：
1. 网络连接问题导致字体文件加载失败
2. 浏览器字体设置冲突
3. 设备系统字体问题
请尝试刷新页面或重启浏览器。
